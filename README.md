# Bullet-Hell

## Descriere

  Un joc inspirat din titluri precum SoulKnight, The Binding of Isaac. Shooter cu view top-down, mouse-aimed, ce se desfasoara intr-un dungeon cu camere generate random si accesibile doar pe rand (accesul liber printr-o camera se deblocheaza dupa eliminarea inamicilor). Ultima camera generata va contine un Boss ce trebuie ucis pentru a castiga jocul. Focusul principal va fi pe evitarea gloantelor inamicilor, HP-ul fiind singura resursa limitata si greu recuperabila.
